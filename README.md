# Challenge My Band / Council Tax Challenge

GitHub-ready React + Vite site for an independent Council Tax band review service.

## Included

- official-style public-information layout
- England 1 April 1991 Council Tax band table
- quick 1991 value estimator
- built-in SVG logo asset
- GOV.UK-inspired visual treatment without implying official government status

## Run locally

```bash
npm install
npm run dev
```

## Notes

- The band table is set up for England, where Council Tax bands are based on 1 April 1991 values.
- The instant estimator is a screening tool only and uses a simplified HPI-style benchmark.
- The site includes an independence notice so the styling does not imply the service is part of GOV.UK or the Valuation Office Agency.
